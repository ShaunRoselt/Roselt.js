About this starter

Build outward from a small, inspectable app shell.

This extra page shows the default file-based route flow in action. Add more files under pages/, keep shared UI in sections/, and move reusable controls into components/.

Back to home

Pages stay direct

Each route is still plain HTML plus optional JS and CSS sidecars, so you can read the app structure straight from the file tree.

Components stay reusable

The custom-button on this page comes from components/custom-button.js, which makes it easy to reuse across routes without copy-pasting markup.

Routing stays simple

This page is available at ?page=about because the file lives at pages/about.html.
