API reference

The public Roselt.js surface is deliberately compact.

Most apps only need Roselt.start(), page modules, and the default folder layout. The table below covers the actual exported surface available in this repository today.

start(options)

The browser-global convenience entry point. In a script-tag setup, window.Roselt.start() forwards to Roselt.start(options) and returns the started app instance.

Browser global helpers

Helper Purpose
Roselt.app Returns the active app instance after start.
Roselt.navigate(target, parameters?, navigationOptions?) Runs programmatic navigation through the active app.
Roselt.href(target, parameters?) Builds an internal route URL using the active app's routing mode.
Roselt.currentUrl() and Roselt.currentRoute() Read the active browser URL and resolved route match.

Roselt.start(options)

Creates a Roselt instance, starts section resolution, starts the router, and returns the app instance.

Option Default Meaning
pageRoot "roselt[page][navigate]" Selector or element used as the routed page root. The default start path requires a roselt[page] element marked with navigate.
routes [] Optional explicit route list. When omitted, Roselt.js resolves page IDs directly from the requested URL and the configured pagesDirectory.
routingMode "query" Controls whether Roselt.js reads a query parameter or the pathname for route resolution.
queryParam "page" The search parameter used when query routing is active.
basePath derived from document.baseURI Base path used by pathname routing and href generation.
components {} Initial component definitions to register before the app starts.
pagesDirectory "pages" Directory Roselt.js uses for automatic page resolution.
sectionsDirectory "sections" Directory Roselt.js uses when resolving shorthand section references.
componentsDirectory "components" Directory used when Roselt.js auto-loads custom element modules by tag name.
defaultPage derived from the page root or home Fallback page when the URL does not explicitly request one.

App instance methods

Method Purpose
start() Starts section resolution, component discovery, router startup, and the initial page render.
stop() Stops router listeners and clears the active global app reference.
resolve(url) Resolves a URL into a route match using the current app options.
renderUrl(url, routeMatch?) Renders a URL directly through the current render engine.
navigate(target, parameters?, navigationOptions?) Performs programmatic navigation using the Navigation API.
href(target, parameters?, sourceUrl?) Builds an internal route URL in the app's configured routing mode.
currentUrl() Returns the current browser URL as a URL object.
currentRoute() Returns the resolved route match for the current URL.
resolveComponent(tagName) Builds the component module path for a custom element tag.
rootHref() Returns the root href for the configured base path.

Page authoring API

Property Meaning
Roselt.Page.Title Sets the document title and the fallback Open Graph/Twitter title text.
Roselt.Page.Description Sets the description and the fallback social description text.
Roselt.Page.Canonical Overrides the canonical URL link for the route.
Roselt.Page.Stylesheets Declares additional stylesheets to load for the page module.
Roselt.Page.Components Registers page-local component definitions for the rendered route.
Roselt.Page.Load Mount hook invoked after the page HTML renders and components are resolved. It receives no required arguments.
Roselt.Page.querySelector() and Roselt.Page.querySelectorAll() DOM query helpers scoped to the active page root for the currently mounted route.
Roselt.Page.route, Roselt.Page.params, and Roselt.Page.notFound Runtime route state for the mounted page, including pathname params and custom 404 details.
Roselt.Page.url and Roselt.Page.query Runtime URL state for the mounted page.
Roselt.Page.href(), Roselt.Page.navigate(), and Roselt.Page.cleanup() Runtime helpers for link building, navigation, and page unmount cleanup registration.
Load return value Return a cleanup function, an object with cleanup or destroy, runtime meta, or null when no teardown is needed.

Named exports from the package entry

Roselt and default export
